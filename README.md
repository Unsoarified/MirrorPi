Through the long-term development of this project, I will develop skills that will serve extremely well in my future as an aspiring engineer as well as fuel
intellectual curiosity. 
Developing a Personalized Smart Mirror seemed like it would be a fun personal project I could soak my time into in hopes developing key skills such as
critial thinking and problem solving through the practice of basic programming and engineering. 

Throughh this project I plan to:
* Self-teach myself a programming language, python, and use it to develop a personalized UI Smart Mirror interface from scratch that sereves to do tasks such as
  - Tell the time, Provide Weather Details, Provide details on the moon phase, perform simple calculations, etc..
    
* Self-teach myself engineering by requiring my self to build the entire frame from scratch as well trying to be as resourceful as possible through the upcycling
of parts such as a Windows 7 LCD Panel that will be used in the Project.

How do I plan to develop this?

- First, I plan to have a simple, but developed code ready to be appled to my smart mirror
(This code will primarily include white texts in order for light to pass through a mirror as well as a black background in ordere to allow for the "Mirror" part 
of the 'Smart Mirror' to stay true.)
- This code will be run through a famous microcontroller known as the Raspberry Pi 4.
(This raspberry pi 4 will be the entire brain of the operation as it will display the code I developed onto the LCD Panel from the Windows 7.)
- A two way mirror will be used in front of the LCD Screen in order for screen to actually have a "reflection" or "mirror attributes". The white text from the code
will pass through the mirror on one side which allows for Data to be displayed on the mirror while also allowing for the mirror to take advantage of the black background
and still be used as originally intended, as a mirror.
-The entire thing will be encased in a self-made wood frame which will have a spot for the raspberry Pi as well as old the mirror and the LCD panel together.
